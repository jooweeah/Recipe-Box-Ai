# Recipe Box

A personal recipe manager where you can save, organize, and rate your favorite recipes.
Built as a Week 8 midterm project exploring AI-assisted full-stack development with
React and Firebase.

## Features
- Add, edit, and delete recipes with ingredients, steps, cook time, and category
- Search recipes by title or ingredient
- Filter recipes by category (breakfast, lunch, dinner, dessert, snack)
- Serving size scaler that adjusts ingredient amounts dynamically
- Rate recipes and toggle "Tried It" status
- User authentication — each user has their own private recipe collection
- Editable user profile with display name and avatar

## Tech Stack
- React (Vite, JavaScript + SWC)
- Firebase Auth + Firestore
- Tailwind CSS
- Deployed on Netlify
